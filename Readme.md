# Kabra Logitech Technical Task

## Setup the Project
# First Clone the project 
```bash
 git clone
```
# To start Backend Run following comands

* first cd to server folder
```bash
cd .\server\
```

* Install All npm packages
```bash
npm i
```

* start the server
```bash
npm run dev
```

# To start Frontend Run following comands
* first cd to server folder
```bash
cd .\client\
```

* Install All npm packages
```bash
npm i
```

* start the server
```bash
npm run dev
```
